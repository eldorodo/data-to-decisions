Common Pitfalls

Chained Indexing

Using df['A']['B'] is ambiguous — pandas may work on a copy. Always use .loc[row, col] for assignments to avoid silent bugs.

# Bad: chained indexing df['col1'][df['col1'] > 0] = 99 # may fail silently # Good: use .loc df.loc[df['col1'] > 0, 'col1'] = 99

SettingWithCopyWarning

Triggered when modifying a DataFrame that might be a slice. Call .copy() explicitly when slicing to own your data.

# Triggers warning: subset = df[df['age'] > 30] subset['score'] = 0 # ⚠️ Warning! # Fix: subset = df[df['age'] > 30].copy() subset['score'] = 0 # OK
