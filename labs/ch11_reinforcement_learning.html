Bellman Optimality Equation

V*(s) = max_a Σ P(s'|s,a)[R(s,a,s') + γV*(s')]
Value Iteration repeatedly applies the Bellman operator until convergence. Each iteration updates all state values using current estimates of neighbors.
V_{k+1}(s) = max_a Σ P(s'|s,a)[R + γ·V_k(s')]
