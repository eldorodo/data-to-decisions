The Degradation Problem

Plain Net Problem: Adding more layers hurts accuracy on training data — not just overfitting. A 56-layer plain net performs worse than a 20-layer plain net. This is the degradation problem.

ResNet Solution: Learn residuals F(x) instead of full mappings H(x). Skip connections create "gradient highways" that preserve gradient magnitude across many layers.
y = F(x, {Wᵢ}) + x

If identity is optimal → F(x) → 0 (easy to learn)
Plain net must learn H(x) = identity (hard)

Skip Connection Diagram

ResNet gradients stay strong because ∂L/∂x = ∂L/∂y · (∂F/∂x + 1). The "+1" means gradient never vanishes completely.

Attention weights
Self-Attention: Each patch computes Query, Key, Value vectors. Attention(Q,K,V) = softmax(QKᵀ/√d)·V. Every patch attends to all others — no local receptive field constraint.

CNN vs ViT Comparison

PropertyCNNViT
Inductive biasTranslation equivariance, localityNone (learns from data)
Receptive fieldGrows with depthGlobal from layer 1

ViT needs more data than CNN to train from scratch but scales better and achieves state-of-the-art with enough data (JFT-300M+). With fine-tuning from pretrained weights, ViT is competitive on smaller datasets.
