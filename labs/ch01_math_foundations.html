Key Takeaways
  • Eigenvectors of the covariance matrix define the principal axes of a data cloud — they point in directions of maximum spread.
  • Eigenvalues tell you how much variance is explained along each eigenvector. λ₁ ≥ λ₂ always (after sorting).
  • PCA dimensionality reduction keeps only the top-k eigenvectors, discarding the rest. This works because low-eigenvalue directions carry little information.
  • The covariance ellipse is fully characterized by the eigenvalue decomposition: C = QΛQᵀ.

Key Takeaways
  • Gradient descent is an iterative first-order optimization algorithm. It requires only gradient information, not second-order curvature.
  • The learning rate η is the most critical hyperparameter: too large → divergence; too small → impractically slow convergence.
  • On elongated loss surfaces (like x² + 5y²), the gradient points "across" the valley rather than "down" it, causing zigzag behaviour — this is why momentum and adaptive methods (Adam, RMSProp) were invented.
  • In deep learning, we use stochastic gradient descent (SGD): compute the gradient on a random mini-batch rather than the full dataset.

Formulas
H(P) = −Σ p·log₂(p)
KL(P‖Q) = Σ p·log₂(p/q)
H(P,Q) = H(P) + KL(P‖Q)
Key Takeaways
  • Shannon entropy H(P) measures average surprise. Uniform distributions have maximum entropy; deterministic ones have zero entropy.
  • KL divergence KL(P‖Q) is not symmetric — it measures the extra bits needed to encode samples from P using a code optimized for Q.
  • Cross-entropy H(P,Q) = H(P) + KL(P‖Q) is used as a loss function in classification. Minimizing it over Q is equivalent to minimizing KL divergence.
  • In ML, we minimize cross-entropy loss between the true label distribution P and the model's predicted distribution Q.
