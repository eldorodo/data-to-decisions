Four components of power analysis:
Effect Size (d): Standardized difference between groups. Larger effects are easier to detect.
Significance Level (α): Probability of a Type I error (false positive). Typically 0.05.
Power (1−β): Probability of detecting a true effect. Aim for 0.80 or higher.
Sample Size (n): Number of observations per group. More data yields more power.
n = ((z_α + z_β) / d)²
In practice: halving the effect size quadruples the required sample size.
